## Run Locally

Clone the project

```bash
  git clone https://github.com/syhrndraa/landing-bc-nextjs
```

Go to the project directory

```bash
  cd landing-bc-nextjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
